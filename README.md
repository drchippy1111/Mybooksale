# BookSale - Simple Online Book Store

A clean, modern, and fully static single-page website for showcasing and selling books. Built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks, no backend required. Perfect for a personal bookshop, side project, or portfolio demo.

Live Demo: [https://mybooksale.vercel.app](https://mybooksale.vercel.app)  
*(Replace with your actual Vercel URL once deployed)*

## ✨ Features

- Responsive design that looks great on mobile, tablet, and desktop
- Clean book listing layout with cover images, titles, descriptions, and prices
- Interactive "Add to Cart" buttons (with simple JavaScript alerts for demo)
- Easy customization — add more books by duplicating HTML sections
- Fast performance: pure static files served via Vercel CDN
- Root URL routing: visits `/` automatically show your book sale page (via `vercel.json`)

## 🖥️ Technologies Used

- **HTML5** — Structure & semantics
- **CSS3** — Styling & responsive layout
- **JavaScript** — Basic interactivity
- **Vercel** — Hosting & auto-deployment
- GitHub — Version control & source hosting

No build tools, no Node.js, no npm — just files you edit and push!
